# 💰 Simple Interest Calculator (Python)

A beginner-friendly Python program that calculates the final balance using the **Simple Interest formula**.
The program validates user input to ensure all values are greater than zero and handles invalid entries gracefully.

---

## 🧮 Formula Used

The program uses the Simple Interest formula:

[
A = P(1 + rt)
]

Where:

* **A** = Final amount
* **P** = Principal amount
* **r** = Rate of interest (per year)
* **t** = Time (in years)

---

## ▶️ How to Run

1. Make sure Python is installed on your system
2. Save the file as `simple_interest.py`
3. Run the program:

```bash id="f2k9as"
python simple_interest.py
```
